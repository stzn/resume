## ■ プロフィール概要

**「作らない機能」を定義できる、実装上がりの PM/コンサルタント**

スタートアップでの**CTO（最高技術責任者）**として経営判断に携わった経験と、大手 IT サービス企業で**テックリード**として大規模開発を牽引した経験を併せ持っています。

私の強みは、**「コードが書ける（工数感がわかる）こと」を武器にした上流工程**です。
単に要望を書き起こすだけでなく、実装視点から「その機能はコストに見合わない」「既存 SaaS で代替すべき」といった**現実的なジャッジ（引き算の提案）**を行うことで、DX プロジェクトの予算超過や炎上を未然に防ぎます。

---

## ■ 提供できる価値（上流工程メイン）

* **プロジェクトの交通整理（要件定義）**
    * ふわっとしたビジネス要件を、エンジニアが実装可能なレベルの「仕様」と「画面遷移」に落とし込みます
* **見積もりセカンドオピニオン**
    * 開発ベンダーからの見積もり・提案内容を技術者視点で診断し、適正価格やリスクをレポートします
* **技術選定・アーキテクチャ設計**
    * AWS/GCP 等のクラウド選定から、アプリか Web かの媒体選定まで、ビジネス規模に最適な構成を設計します
